akSlider
========

WORK IN PROGESS!!!

Lightweight, object oriented, jQuery based slider

Options
-----

#### showArrows
##### Default
	onhover
##### Other Values
	always

#### showNavButtons
##### Default
	always
##### Other Values
	onhover

Usage
-----

	$('.tutorial-slideshow').akSlider({
		slides: {
			1: {
				html: '<div>Any HTML you want!</div>'
			},
			2: {
				image: '/path/to/image.png'
			}
		}
	    // other options here
	});
